# OpenSkills -- Universal Skills Loader (10.6K stars)

`npm i -g openskills` -- Load skills for any AI coding agent.

## What It Is
OpenSkills is a universal skills loader that works with Claude Code, Codex CLI, Cursor, and any AI coding agent.

```bash
npm install -g openskills

# Load skills from any source
openskills load github:vercel-labs/agent-skills
openskills load github:cloudflare/skills
openskills load github:anthropics/skills

# Search available skills
openskills search react
openskills search cloudflare
```
