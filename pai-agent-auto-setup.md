# PAI Agent Auto-Setup

Master skill for deploying the complete PAI stack.

## What It Sets Up
1. **Vercel** -- Website deployment
2. **OpenCode / Claude Code / Codex** -- Coding agents
3. **Ollama** -- Local models
4. **Ghost.Build** -- Agent DB
5. **Jam** -- Voice rooms
6. **Here.Now** -- Publishing
7. **CUA** -- Computer use
8. **cmux** -- Multi-screen workspace

## One-Click Install
```bash
curl -fsSL https://pai.build/deploy.sh | bash
```

- `scripts/install.sh` -- Full installer
